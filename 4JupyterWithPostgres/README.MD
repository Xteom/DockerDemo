# Jupyter Notebook with PostgreSQL using Docker Compose

This is an example of creating a Jupyter Notebook environment with a PostgreSQL database using Docker Compose.

## Requirements

- Docker
- Docker Compose

## Getting Started

1. Start the Jupyter Notebook and PostgreSQL services by running `make up`.
2. Open the displayed JupyterLab URL in your browser.
3. To stop the services, run `make down`.

## Makefile Commands

- `make up`: Start the Jupyter Notebook and PostgreSQL services.
- `make down`: Stop the Jupyter Notebook and PostgreSQL services.
